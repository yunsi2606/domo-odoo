HR Planning =========== Complete employee shift planning and scheduling module for Odoo 18 Community. This module provides all the features of the Enterprise Planning module: **Features:** * Planning slots with Gantt, Calendar, Kanban, and List views * Role-based shift assignments * Shift templates for quick scheduling * Recurring/repeating shifts * Conflict detection for overlapping shifts * Time off integration * Employee self-service * Publish and notify employees * Copy previous week scheduling * Planning analytics and reports **Main Views:** * Gantt chart for visual scheduling * Calendar view for date-based planning * Kanban for status-based workflow * List view for bulk operations * Pivot/Graph for analysis
